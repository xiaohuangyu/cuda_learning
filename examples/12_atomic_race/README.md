# 第十二章示例：原子操作与竞争条件

本目录包含第十二章配套的示例代码，演示竞争条件和原子操作。

## 文件说明

| 文件 | 描述 |
|------|------|
| `01_race_condition.cu` | 竞争条件演示 - 错误的并行累加 |
| `02_atomic_add.cu` | 使用原子操作修复竞争条件 |
| `03_atomic_performance.cu` | 原子操作性能对比 |
| `04_warp_reduce.cu` | Warp级别规约优化 |
| `05_block_reduce.cu` | Block级别规约优化 |
| `06_atomic_cas.cu` | atomicCAS实现自定义原子操作 |
| `07_histogram.cu` | 实战案例：并行直方图 |

## 编译运行

```bash
# 编译所有示例
make all

# 运行单个示例
./01_race_condition
./02_atomic_add

# 使用 ncu 分析性能
ncu ./03_atomic_performance
```

## 学习要点

1. **竞争条件**：理解为什么普通并行累加会出错
2. **原子操作**：掌握 `atomicAdd` 等原子函数的使用
3. **性能代价**：理解原子操作导致的串行化问题
4. **优化策略**：学习通过规约减少原子操作次数

## 性能对比

| 实现方式 | 正确性 | 相对性能 |
|----------|--------|----------|
| 直接累加 | ❌ 错误 | 基准 |
| 原子加法 | ✅ 正确 | 很慢 |
| Warp规约 | ✅ 正确 | 较快 |
| Block规约 | ✅ 正确 | 最快 |
