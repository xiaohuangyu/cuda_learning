# 第二十章示例：卷积算子实现

本目录包含第二十章配套的示例代码，演示CUDA卷积算子的实现与优化。

## 文件说明

| 文件 | 描述 |
|------|------|
| `01_direct_conv.cu` | 直接卷积实现（Naive版本） |
| `02_im2col_conv.cu` | im2col + GEMM卷积实现 |
| `03_smem_conv.cu` | 共享内存优化卷积 |

## 编译运行

```bash
# 编译
make all

# 运行
./01_direct_conv
./02_im2col_conv
./03_smem_conv
```

## 卷积基础

### 直接卷积
```
输出[y][x] = sum over (ky, kx) {
    输入[y+ky][x+kx] * 卷积核[ky][kx]
}
```

### im2col方法
将卷积转换为矩阵乘法：
1. 将输入按滑动窗口展开成矩阵（im2col）
2. 用GEMM计算矩阵乘法
3. 结果即为卷积输出

### 共享内存优化
- 将输入分块加载到共享内存
- 减少全局内存访问次数
- 提高数据复用率
